impromtu
===========

A sample app created for the purpose of illustrating unit tests and test-driven development.

Setup
============

As simple as:

    git clone git://github.com/neweryankee/nyw_tdd.git

Sample Usage
============

    $ cd nyw_tdd/
    $ bin/impromptu 3/15/1979

Running tests
============

    $ cd nyw_tdd/
    $ ruby test/schedule_test.rb
